<li><H3>Project Overview</H3>
This project explores whether winning the toss in the Indian Premier League (IPL) provides a significant advantage. Using Tableau, I developed an interactive dashboard to visualize win percentages based on toss outcomes, team decisions (bat vs. field), and chasing vs. defending strategies.
</li>
<li><H3>Key Insights</H3>
<ol>
<li><H4>Toss Impact:</H4> Winning the toss gives a slight edge, with a league-wide win rate of 50.56%.</li>

<li><H4>Chasing vs. Defending:</H4> There is a clear preference for chasing, which yields a 53.12% win rate compared to 44.91% for defending.</li>

<li><H4>Top Performers:</H4> Gujarat Titans have the highest success rate after winning the toss at 61.67%.</li>
</ol>
